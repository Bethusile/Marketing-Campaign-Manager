# Marketing Client

This project is a React + TypeScript app bootstrapped with Vite. It uses Material UI (MUI) for UI components and React Router for routing.

## Setup Instructions

1. **Clone the repository**

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Create your environment file**
   - Copy `.env.example` to `.env`:
   - Edit `.env` and set your `SERVER_URL` value.

4. **Start the development server**
   ```sh
   npm run dev
   ```

## Project Structure


```
src/
   api/          # API request logic 
   components/   # Reusable React components 
   pages/        # Top-level route pages
   utils/        # Utility/helper functions used throughout the app
   assets/       # Static assets
   styles.css    # Custom project-wide styles
   App.tsx       # Main app component, sets up theme and routing
   main.tsx      # Entry point for React
public/         # Static public files
```
